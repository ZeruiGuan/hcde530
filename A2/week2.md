# Week 2

## C2: Code Literacy and Documentation

This week I worked with a Python script that reads a CSV file, processes each row, and counts the number of words in participant responses. I focused on understanding how the code uses a loop to go through the data and how the count_words function works.

I added inline comments in plain English to explain how the CSV is read, how responses are stored, and how the results are calculated and printed. This demonstrates my ability to read, interpret, and document code clearly.
